# react-typeahead
A website for people who want to see the suggestions based on the letter they type in.
# Usage
```
$ node index.js
```
# Use
* HTML5
* CSS3
* React
* React-DOM
* Babel
* Babelify
* watchify
* browserify
* Modern JavaScript (ES2015+)
* MongoDB
* Express
* Node.js
